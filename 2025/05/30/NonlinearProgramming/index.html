非线性规划——模型引出

除了线性规划模型，优化类问题往往还有一些更复杂的

  • 常见的收益率、传播率、经济增长率等规划问题，通常涉及 \(\frac{1}{x}\) 形式的表达，属于典型的非线性问题。

  • 空间运动问题（如空间约束、避障、角度调整等），由于涉及曲线轨迹和角度变化，常用到三角函数、指数函数等非线性表达，因而也属于非线性问题的范畴。

  • 运输优化问题，例如已知多个坐标点，需确定最优运输路径。此类问题通常用到欧几里得距离公式，其数学形式包含变量的平方和开方，因此也属于非线性问题。

  • 示例：

[ \[\begin{aligned} &\min~y = x_1^2 + x_2^2 - x_1x_2 - 2x_1 - 5x_2, \\ &\text{s.t.} \begin{cases} (-x_1 - 1)^2 + x_2 \geq 0, \\ 2x_1 - 3x_2 + 6 \geq 0 \end{cases} \end{aligned}\]

]

  • 这就是一个非线性规划模型，与线性规划类似，它同样需要在确定决策变量的基础上，构建目标函数并列出相应的约束条件。

非线性规划——模型引出

非线性规划简介

非线性规划是运筹学的重要分支，致力于求解目标函数或约束条件包含非线性函数的最优化问题。20世纪50年代初，库恩（H.W. Kuhn）和塔克（A.W. Tucker）提出的基本定理（K-T条件）为该领域奠定了理论基础。该方法在工业、交通运输、经济管理与军事等领域应用广泛，尤其在最优设计中提供关键的数学基础与计算方法，具有显著的实用价值。

非线性规划模型特点

  • 非线性特征： 模型中至少包含一个非线性项，例如 ( x^2 )、( e^x )、( )、( x )、( _2 x ) 等形式。
  • 与线性规划的区别：
    • 线性规划 (LP)： 存在通用的精确求解方法（如单纯形法），其最优解必然位于可行域的边界上。
    • 非线性规划 (NLP)： 最优解（若存在）可能出现在可行域的内部或边界任意点。目前尚无适用于所有 NLP 问题的通用解法，不同算法有其特定的适用范围。
  • 数学建模中的应用： 对于数学建模实践，掌握利用 MATLAB 等工具的内置函数求解非线性规划问题（通常得到近似最优解）即可满足需求。

非线性规划——模型原理

非线性规划模型的标准型

\[\begin{aligned} &\min ~f(x)\\ &\text{s.t.}\begin{cases} Ax \leq b,~Aeq\cdot x = beq&(\text{线性})\\ c(x) \leq 0,~Ceq(x) = 0&(\text{非线性})\\ lb \leq x \leq ub \end{cases} \end{aligned}\]

非线性规划求解的Matlab函数

  • fmincon函数: [x, fval] = fmincon ( fun, x0, A, b, Aeq, beq, lb, ub, nonlcon, option)

    • ( fun )：把目标函数定义为一个单独的函数文件（min）
    • ( x0 )：决策变量的初始值
    • ( A, b )：线性约束的不等式变量系数矩阵和常数项矩阵（<或≤）
    • ( Aeq, beq )：线性约束的等式变量系数矩阵和常数项矩阵
    • ( lb, ub )：决策变量的最小取值和最大取值
    • ( nonlcon )：非线性约束，包括不等式和等式
    • ( option )：求解非线性规划使用的方法

  • 非线性规划中对于初始值( x0 )的选取非常重要，因为非线性规划的算法求解出来的是一个局部最优解。如果要求全局最优解，有两个思路：

    • 给定不同的初始值，在里面找到一个最优解；
    • 先用蒙特卡罗模拟，得到一个蒙特卡罗解，然后将这个解作为初始值求最优解

  • options选项可以给定求解的算法，一共有五种：interior-point（内点法）、trust-region-reflective（信赖域反射法）、sqp（序列二次规划法）、sqp-legacy（约束非线性优化算法）、active-set（有效集法）。不同的算法有其各自的优缺点和适用情况，我们可以改变求解的算法来对比求解的结果。

  • fun表示目标函数，我们要编写一个独立的“m文件”储存目标函数 function f = fun(x) f = …… end

  • nonlcon 表示非线性部分的约束，也要编写一个独立的“m文件”储存非线性约束条件
    \[\begin{aligned} &function ~~[c, ceq] = nonlfun(x)\\ &\qquad c=[非线性不等式约束1;\\ &\qquad\qquad\vdots\\ &\qquad 非线性不等式约束n]\\ &\qquad ceq=[非线性等式约束1;\\ &\qquad\qquad\vdots\\ &\qquad 非线性等式约束m]\\ &end \end{aligned}\]

  • 决策变量下标要改为括号，比如 ( x_1 ) 要改为 ( x(1) )，matlab才能识别

  • 若不存在某种约束，可以用“[]”替代，若后面全为“[]”且option使用默认，后面的“[]”可省略

非线性规划——典型例题

选址问题

  • 临时料场：( A(5,1) )，( B(2,7) )；日储量各20吨。
工地位置坐标及日需求量 I II III IV V VI
横坐标 1.25 8.75 0.5 5.75 3 7.25
纵坐标 1.25 0.75 4.75 5 6.5 7.25
日需求量 3 5 4 7 6 11
  • （1）试制定每天的供应计划，即从两料场分别向各工地运送多少吨水泥，使总的吨千米数最小？
  • （2）为了进一步减少吨千米数，打算舍弃两个临时料场，改建两个新的，日储量各为20吨，问应建在何处，节省的吨千米数为多大？

建立模型

  • 确定决策变量
    设第\(i\)个工地的坐标 ((a_i, b_i))，水泥日用量 (d_i)，(i = 1, 2, ..., 6)；料场位置 ((x_j, y_j))，日储量 (e_j)，(j = 1, 2)；从料场向工地的运送量为 (x_{ij})

  • 确定约束条件
    料场水泥运输总量不超过其日储量：({i=1}^6 x{ij} e_j, j = 1, 2)

    两个料场向某工地运输量之和等于该工地水泥日用量：({j=1}^2 x{ij} = d_i, i = 1, 2, ..., 6)

  • 3）确定目标函数
    求总吨千米数最小，即运送量乘运送距离求和最小 (min f = {j=1}^2 {i=1}^6 x_{ij} )

  • 4）建立模型
    [ min f = {j=1}^2 {i=1}^6 x_{ij} ]
    [ s.t.

    \[\begin{cases} \sum_{i=1}^6 x_{ij} \leq e_j, j = 1, 2 \\ \sum_{j=1}^2 x_{ij} = d_i, i = 1, 2, ..., 6 \\ x_{ij} \geq 0, i = 1, 2, ..., 6; j = 1, 2 \end{cases}\]

    ]

非线性规划——典型例题

选址问题

  • 5）求解
    ✓ 对于第一问：因料场位置已知，故决策变量仅为 ( x_{ij} )，为线性规划模型
    ✓ 对于第二问：新料场位置未知，所以 ( x_j ) 和 ( y_j ) 均为变量，且不是线性的，故为非线性规划模型

    ✓ 共有8个约束
    [ min f = {j=1}^2 {i=1}^6 x_{ij} ] [ s.t.

    \[\begin{cases} \sum_{i=1}^6 x_{ij} \leq e_j, j = 1, 2 \\ \sum_{j=1}^2 x_{ij} = d_i, i = 1, 2, \ldots, 6 \\ x_{ij} \geq 0, i = 1, 2, \ldots, 6; j = 1, 2 \end{cases}\]

    ]

    ✓ 注意，在 Matlab 里这些双角标的变量要改为单角标的变量，如 ( x_{11} x_1, x_{21} x_2, , x_{62} x_{12} )

%% 使用蒙特卡罗的方法来求初始值（推荐） clc, clear; n=10000000; %生成的随机数组数 x1=unifrnd(-100,100,n,1); %生成在[-100,100]之间均匀分布的随机数组 x2=unifrnd(-100,100,n,1); fmin=+inf; %初始化函数的最小值为正无穷 for i=1:n x = [x1(i), x2(i)]; if ((x(1)-1)^2-x(2)<=0) & (-2x(1)+3x(2)-6 <= 0) %判断是否满足条件 result = -x(1)2-x(2)2 +x(1)x(2)+2x(1)+5*x(2); if result < fmin fmin = result; x0 = x; end end end disp('蒙特卡罗选取的初始值为: '); disp(x0) A = [-2 3]; b = 6; [x, fval] = fmincon(@fun1,x0,A,b,[],[],[],[],@non1fun1) fval = -fval

function f = fun1(x) % 目标函数 % 输入x为决策变量向量[x1,x2] % 输出f为目标函数值 % max f(x) = x1^2 + x2^2 - x1x2 - 2x1 - 5x2 f = -x(1)2-x(2)2 + x(1)x(2)+2x(1)+5x(2); end

function [c, ceq] = nonlfun1(x) % 非线性约束函数 % 输入x为决策变量向量[x1,x2] % 输出c为非线性不等式约束，ceq为非线性等式约束 % -(x1-1)^2 + x2 >= 0 c = [(x(1)-1)^2 - x(2)]; ceq = []; % 无非线性等式约束 end
