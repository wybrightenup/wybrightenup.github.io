高等数学知识点总结

向量代数与空间解析几何及多元微分学在几何上的应用

第一节 向量代数

一、数量积

  1. 几何表示：$\vec{a}\cdot\vec{b}=|\vec{a}||\vec{b}|\cos \theta$
  2. 代数表示：$\vec{a}\cdot\vec{b}=a_x b_x+a_y b_y+a_z b_z$
  3. 运算规律
    (1) 交换律：$\vec{a}\cdot\vec{b}=\vec{b}\cdot\vec{a}$
    (2) 分配律：$\vec{a}\cdot(\vec{b}+\vec{c})=\vec{a}\cdot\vec{b}+\vec{a}\cdot\vec{c}$
  4. 几何应用
    (1) 求模：$|\vec{a}|=\sqrt{\vec{a}\cdot\vec{a}}$
    (2) 分配律：求夹角：$\cos\theta=\frac{\vec{a}\cdot \vec{b}}{|\vec{a}||\vec{b}|}$
    (3) 分配律：判定两向量垂直：$\vec{a}\perp\vec{b}\Leftrightarrow\vec{a}\cdot\vec{b}=0$

二、向量积

  1. 几何表示：$\vec{a}\times\vec{b}$ 是一向量
    (1) 模：$|\vec{a}\times\vec{b}|=|\vec{a}||\vec{b}|\sin\theta$
    (2) 方向：右手法则
  2. 代数表示
    $$\vec{a}\times\vec{b}=\begin{vmatrix}\vec{i}&\vec{j}&\vec{k}\a_x&a_y&a_z\b_x&b_y&b_z\end{vmatrix}$$
  3. 运算规律
    (1) $\vec{a}\times\vec{b} = -(\vec{b} \times \vec{a})$
    (2) 分配律：$\vec{a}\times(\vec{b}+\vec{c})=\vec{a}\times\vec{b}+\vec{a}\times\vec{c}$
  4. 几何应用
    (1) 求同时垂直于 $\vec{a}$ 和 $\vec{b}$ 的向量：$\vec{a}\times\vec{b}$
    (2) 求以 $\vec{a}$ 和 $\vec{b}$ 为邻边的平行四边形面积：$S=|\vec{a}\times\vec{b}|$
    (3) 判定两向量平行：$\vec{a}\parallel\vec{b}\Leftrightarrow\vec{a}\times\vec{b}=0$

三、混合积

  1. 代数表示
    $$(\vec{a}\vec{b}\vec{c})=\begin{vmatrix}a_x&a_y&a_z\b_x&b_y&b_z\c_x&c_y&c_z\end{vmatrix}$$
  2. 混合积的运算规律
    (1) 轮换对称性：$(\vec{a}\vec{b}\vec{c}) = (\vec{b}\vec{c}\vec{a}) =(\vec{c}\vec{a}\vec{b})$
    (2) 交换变号：$(\vec{a}\vec{b}\vec{c})= -(\vec{a}\vec{c}\vec{b})$
  3. 混合积的几何应用
    (1) 平行六面体的体积：$V_{\text{平行六面体}}=|(\vec{a}\vec{b}\vec{c})|$
    (2) 判定三向量共面：$\vec{a}, \vec{b}, \vec{c}$ 共面 $\Leftrightarrow(\vec{a}\vec{b}\vec{c})=0$

第二节 空间平面与直线

一、平面方程

  1. 一般式：$Ax + By + Cz + D = 0 $，法向量 $\vec{n} =(A, B, C)$.
  2. 点法式：$A(x - x_0) + B(y - y_0) + C(z - z_0) = 0 $.
  3. 三点式：$\begin{vmatrix}x-x_1&y-y_1&z-z_1\x_2-x_1&y_2-y_1&z_2-z_1\x_3-x_1&y_3-y_1&z_3-z_1\end{vmatrix}=0$.
  4. 截距式：$\frac{x}{a} + \frac{y}{b} + \frac{z}{c} = 1$. （$a\neq0,b\neq0,c\neq0$）
  5. 平面束：过直线$\begin{cases}A_1x + B_1y + C_1z + D_1 = 0 \A_2x + B_2y + C_2z + D_2 = 0\end{cases}$的所有平面
    $$\lambda(A_1x + B_1y + C_1z + D_1) +\mu(A_2x + B_2y + C_2z + D_2) = 0.$$
  6. 点到平面的距离：点 $(x_0, y_0, z_0)$ 到平面 $Ax+By+Cz+D=0$ 的距离为
    $$d = \frac{|Ax_0 + By_0 + Cz_0 + D|}{\sqrt{A^2 + B^2 + C^2}}.$$

二、直线方程

  1. 一般式：
    $$\begin{cases}
    A_1x + B_1y + C_1z + D_1 = 0, \
    A_2x + B_2y + C_2z + D_2 = 0.
    \end{cases}
    $$
  2. 对称式：$\frac{x - x_0}{l} = \frac{y - y_0}{m} = \frac{z - z_0}{n}$.
  3. 参数式：$x = x_0 + lt, y = y_0 + mt, z = z_0 + nt $.
  4. 平面与直线的位置关系：关键是通过平面的法线向量和直线的方向向量来判断平行、垂直、夹角等关系.
  5. 点到直线的距离：
    点 $(x_0, y_0, z_0)$ 到直线 $\frac{x - x_1}{l} = \frac{y - y_1}{m} = \frac{z - z_1}{n}$ 的距离公式为：
    $$d = \frac{\left| (x_1 - x_0, y_1 - y_0, z_1 - z_0) \times (l, m, n) \right|}{\sqrt{l^2 + m^2 + n^2}}$$

第三节 曲面与空间曲线

一、曲面方程

  1. 一般式：$F(x,y,z)=0$ 或 $z=f(x,y)$.

  2. 常见曲面
    (1) 旋转面：一条平面曲线绕其平面上一条直线旋转一周所成的曲面: 设 $L$ 是 $Oyz$ 平面上一条曲线，其方程为：$\begin{cases}f(y,z) = 0 \x = 0\end{cases}$.

    • $L$ 绕 $y$ 轴旋转所得旋转面方程为 $f(y,\pm\sqrt{x^2+z^2})=0$.
    • $L$ 绕 $z$ 轴旋转所得旋转面方程为 $f(\pm\sqrt{x^2+y^2},z)=0$.

    (2) 柱面： 平行于定直线并沿定曲线 $\Gamma$ 移动的直线 $L$ 形成的轨迹.

    • 准线为 $\Gamma:\begin{cases} f(x,y) = 0\ z = 0\end{cases}$母线平行于 $z$ 轴的柱面方程为 $f(x,y) = 0 $.
    • 准线为 $\Gamma:\begin{cases} F(x,y,z) = 0\ G(x,y,z) = 0\end{cases}$ 母线平行于 $z$ 轴的柱面方程为 $F(x,y,z)=0 $ 和 $G(x,y,z) = 0 $，联立消去 $z$ 所得的二元方程 $H(x,y) = 0 $.

    (3) 二次曲面

    • 椭圆锥面：$\frac{x^2}{a^2}+\frac{y^2}{b^2}=z^2$；特别地，圆锥面：$x^2+y^2=z^2$.
    • 椭球面：$\frac{x^2}{a^2}+\frac{y^2}{b^2}+\frac{z^2}{c^2}=1$；特别地，球面：$x^2+y^2+z^2=R^2$.
    • 单叶双曲面：$\frac{x^2}{a^2}+\frac{y^2}{b^2}-\frac{z^2}{c^2}=1$.
    • 双叶双曲面：$\frac{x^2}{a^2}-\frac{y^2}{b^2}-\frac{z^2}{c^2}= 1$.
    • 椭圆抛物面：$\frac{x^2}{a^2} + \frac{y^2}{b^2} = z$；特别地，旋转抛物面 $z=x^2+y^2$.
    • 双曲抛物面（马鞍面）：$\frac{x^2}{a^2}-\frac{y^2}{b^2}=z$.

二、空间曲线

  1. 参数式：
    $$\begin{cases}
    x = x(t) \
    y = y(t) \
    z = z(t)
    \end{cases}$$
  2. 一般式：
    $$\begin{cases}
    F(x,y,z) = 0 \
    G(x,y,z) = 0
    \end{cases}$$
  3. 空间曲线投影：
    (1) 曲线 $\Gamma: \begin{cases} F(x,y,z) = 0 \ G(x,y,z) = 0 \end{cases}$ 消去 $z$ 得到关于 $xOy$ 面的投影柱面 $H(x,y) = 0$.
    (2) 曲线 $\Gamma$ 在 $xOy$ 面上的投影曲线方程为 $\begin{cases} H(x,y) = 0\ z = 0 \end{cases}$.

第四节 多元函数微分在几何上的应用

一、 曲面的切平面与法线

  1. 曲面 $F(x,y,z)=0$： 法向量为 $\vec{n}=(F_x,F_y,F_z)$.
  2. 曲面 $z=f(x,y)$：法向量为 $\vec{n}=(f_x,f_y,-1)$.

二、曲线的切线与法平面

  1. 参数式曲线：
    $$\begin{cases}
    x = x(t), \
    y = y(t), \
    z = z(t),
    \end{cases}$$
    切向量为 $\tau=(x’(t_0),y’(t_0),z’(t_0)) $.

  2. 一般式曲线：
    $$\begin{cases}
    F(x,y,z) = 0, \
    G(x,y,z) = 0,
    \end{cases}$$
    切向量为 $\vec{\tau}=\vec{n_1}\times\vec{n_2}$，其中 $\vec{n_1}=(F_x,F_y,F_z),\vec{n_2}=(G_x,G_y,G_z)$.

第五节 方向导数与梯度

  1. 方向导数
    (1) 定义：$\left.\frac{\partial f}{\partial l}\right|{(x{0},y_{0})} = \lim\limits_{t \to 0^+} \frac{f(x_0 + t \cos \alpha, y_0 + t \cos \beta) - f(x_0, y_0)}{t}$
    (2) 计算：若 $z=f(x,y)$ 可微，则 $\frac{\partial f}{\partial l} = \frac{\partial f}{\partial x} \cos \alpha + \frac{\partial f}{\partial y} \cos \beta$
  2. 梯度：
    (1) 定义：设函数 $z=f(x,y)$ 有连续一阶偏导数，则函数 $z=f(x,y)$ 在点 $(x,y)$ 处的梯度为 $\textbf{grad } z = \frac{\partial f}{\partial x}\vec{i} + \frac{\partial f}{\partial y}\vec{j}$
    (2) 注：梯度 $\textbf{grad } z$ 是一个向量，它的方向是函数在这点方向导数最大的方向，它的模等于方向导数的最大值.

多元函数微分学

第一节 重极限、连续、偏导数、全微分（概念、理论）

一、重极限

定义 设函数 $ f(x,y) $ 在区域 $ D $ 上有定义，点 $ P_0 (x_0, y_0) \in D $ 为 $ D $ 的聚点，如果 $ \forall \varepsilon > 0 $，存在 $ \delta > 0 $，当 $ P(x,y) \in D $，且 $ 0 < \sqrt{(x - x_0)^2 + (y - y_0)^2} < \delta $ 时，都有
$$|f(x,y) - A| < \varepsilon $$ 成立，则称常数 $ A $ 为函数 $ f(x,y) $ 当 $ (x,y) \to (x_0, y_0) $ 时的极限，记为
$$\lim\limits_{(x,y) \to (x_0, y_0)} f(x,y) = A \quad \text{或} \quad \lim\limits_{\substack{x \to x_0\y\to y_0}} f(x,y) = A \quad \text{或} \quad \lim\limits_{P \to P_0} f(P) = A.$$注

  1. 这里的极限是要求点 $ (x,y) $ 在 $ D $ 内以任意方式趋近于点 $ (x_0, y_0) $ 时，函数 $ f(x,y) $ 都趋近于同一确定的常数 $ A $，否则该极限就不存在。
  2. 一元函数极限中的下述性质对多元函数仍成立
    ① 局部有界性；
    ② 保号性；
    ③ 有理运算；
    ④ 极限与无穷小的关系；
    ⑤ 夹逼性。

【例】证明下列重极限不存在

(1) $\lim\limits_{x \to 0} \frac{xy}{x^2 + y^2}$; (2) $\lim\limits_{x \to 0} \frac{xy^2}{x^2 + y^4}$.

证明 (1) 取直线 $y = kx$, 让点 $(x,y)$ 沿直线 $y = kx$ 趋于 $(0,0)$ 点, 此时有

$$\lim\limits_{\substack{y=kx\ x\to 0}} \frac{xy}{x^2 + y^2} = \lim\limits_{x \to 0} \frac{kx^2}{x^2 + k^2 x^2} = \frac{k}{1 + k^2}$$

则重极限 $\lim\limits_{\substack{x \to 0\ y\to 0}} \frac{xy}{x^2 + y^2}$ 不存在.

注 本题中的方法是证明重极限不存在的常用方法.

(2) 取直线 $y = kx$, 则

$$\lim\limits_{\substack{y=kx\ x\to 0}} \frac{xy^2}{x^2 + y^4} = \lim\limits_{x \to 0} \frac{k^2 x^3}{x^2 + k^4 x^4} = \lim\limits_{x \to 0} \frac{k^2 x}{1 + k^4 x^2} = 0$$

这说明沿任何一条过原点的直线 $y = kx$ (不包括y轴)趋于 $(0,0)$ 点时, 极限存在且都为零, 并且若沿y轴趋于 $(0,0)$ 点极限也为零, 事实上

$$\lim\limits_{\substack{x=0\ y\to 0}} \frac{xy^2}{x^2 + y^4} = 0$$

这能否说明重极限 $\lim\limits_{x \to 0} \frac{xy^2}{x^2 + y^4}$ 存在且为零呢? 不能! 事实上若沿过原点的抛物线 $x = y^2$ 趋于 $(0,0)$ 点时, 就有

$$\lim\limits_{\substack{x=y^2\y\to 0}} \frac{xy^2}{x^2 + y^4} = \lim\limits_{y \to 0} \frac{y^4}{y^4 + y^4} = \frac{1}{2}$$

故极限 $\lim\limits_{\substack{x \to 0\y\to 0}} \frac{xy^2}{x^2 + y^4}$ 不存在.

注 证明重极限不存在的常用方法: 沿两种不同路径得到的极限不同（通常可取过点 $(x_0, y_0)$ 的直线）.

二、连续

定义 若 $\lim\limits_{\substack{x \to x_0\y\to y_0}} f(x,y) = f(x_0, y_0)$，则称 $f(x,y)$ 在点 $(x_0, y_0)$ 处连续。
性质

  1. 连续函数的和、差、积、商（分母不为零）及复合仍连续；
  2. 基本初等函数在其定义域内连续；初等函数在其定义区域内连续；
  3. 有界闭区域上连续函数的性质
    ① 有界性：若 $f(x,y)$ 在有界闭区域 $D$ 上连续，则 $f(x,y)$ 在 $D$ 上有界。
    ② 最值性：若 $f(x,y)$ 在有界闭区域 $D$ 上连续，则 $f(x,y)$ 在 $D$ 上必有最大值和最小值。
    ③ 介值性：若 $f(x,y)$ 在有界闭区域 $D$ 上连续，则 $f(x,y)$ 在 $D$ 上可取到介于最小值与最大值之间的任何值。

【例】判断函数 $f(x,y) =
\begin{cases}
\frac{xy}{\sqrt{x^2 + y^2}}, & (x,y) \neq (0,0)\
a, & (x,y) = (0,0)
\end{cases}$ 的连续性。

解：因为 $0 \leq \left| \frac{xy}{\sqrt{x^2 + y^2}} \right| \leq |y|$，则 $\lim\limits_{\substack{x \to 0\y\to 0}} \frac{xy}{\sqrt{x^2 + y^2}} = 0$。

若 $a = 0$，则 $f(x,y)$ 处处连续；若 $a \neq 0$，则 $f(x,y)$ 除点 $(0,0)$ 外处处连续。

三、偏导数

1. 定义

$$f_x(x_0, y_0) = \lim\limits_{\Delta x \to 0} \frac{f(x_0 + \Delta x, y_0) - f(x_0, y_0)}{\Delta x} = \frac{\text{d}}{\text{d}x} f(x, y_0) \bigg|_{x=x_0},$$

$$f_y(x_0, y_0) = \lim\limits_{\Delta y \to 0} \frac{f(x_0, y_0 + \Delta y) - f(x_0, y_0)}{\Delta y} = \frac{\text{d}}{\text{d}y} f(x_0, y) \bigg|_{y=y_0}.$$

注 $f_x(x_0, y_0)$ 就是一元函数 $f(x, y_0)$ 在 $x_0$ 处的导数；$f_y(x_0, y_0)$ 就是一元函数 $f(x_0, y)$ 在 $y_0$ 处的导数。

【例】设$f(x,y) = x + 2y + (y-1) \arcsin \frac{x}{1+xy}$，求 $f_x(0,1)$ 和 $f_y(0,1)$。

解：$f_x(0,1) = \frac{\text{d}}{\text{d}x} f(x,1) \big|{x=0} = \frac{\text{d}}{\text{d}x} (x+2) \big|{x=0} = 1,$

$f_y(0,1) = \frac{\text{d}}{\text{d}y} f(0, y) \big|{y=1} = \frac{\text{d}}{\text{d}y} (2y) \big|{y=1} = 2.$

2. 几何意义

$f_x(x_0, y_0)$ 表示曲线 $z = f(x, y_0)$ 在点 $(x_0, y_0, f(x_0, y_0))$ 处的切线对 $x$ 轴的斜率；
$f_y(x_0, y_0)$ 表示曲线 $z = f(x_0, y)$ 在点 $(x_0, y_0, f(x_0, y_0))$ 处的切线对 $y$ 轴的斜率。

3. 高阶偏导数

设 $z = f(x, y)$，则
$$\frac{\partial^2 z}{\partial x^2} = f_{xx}(x, y) = \frac{\partial}{\partial x} \left( \frac{\partial z}{\partial x} \right),$$

$$\frac{\partial^2 z}{\partial x \partial y} = f_{xy}(x, y) = \frac{\partial}{\partial y} \left( \frac{\partial z}{\partial x} \right),$$

$$\frac{\partial^2 z}{\partial y \partial x} = f_{yx}(x,y) = \frac{\partial}{\partial x} \left( \frac{\partial z}{\partial y} \right), \quad \frac{\partial^2 z}{\partial y^2} = f_{yy}(x,y) = \frac{\partial}{\partial y} \left( \frac{\partial z}{\partial y} \right).$$

定理 如果函数 $z = f(x,y)$ 的两个二阶混合偏导数 $f_{xy}(x,y)$ 及 $f_{yx}(x,y)$ 在区域 $D$ 内连续，则在区域 $D$ 内恒有
$$f_{xy}(x,y) = f_{yx}(x,y).$$

四、全微分

1. 定义

若 $\Delta z = f(x_0 + \Delta x, y_0 + \Delta y) - f(x_0, y_0) = A\Delta x + B\Delta y + o(\rho)$, 则称函数 $z = f(x,y)$ 在点 $(x_0, y_0)$ 可微, $A\Delta x + B\Delta y$ 称为函数 $z = f(x,y)$ 在点 $(x_0, y_0)$ 的全微分, 记为
$$\text{d}z = A\Delta x + B\Delta y.$$

注 以下四条等价
(1) $\Delta z = f(x_0 + \Delta x, y_0 + \Delta y) - f(x_0, y_0) = A\Delta x + B\Delta y + o(\rho)$;
(2) $$\lim\limits_{\substack{\Delta x\to0\\\Delta y\to0}} \frac{[f(x_0 + \Delta x, y_0 + \Delta y) - f(x_0, y_0)] - (A\Delta x + B\Delta y)}{\sqrt{(\Delta x)^2 + (\Delta y)^2}} = 0;$$
(3) $\Delta z = f(x,y) - f(x_0, y_0) = A(x - x_0) + B(y - y_0) + o(\rho)$;
(4) $$\lim\limits_{\substack{x \to x_0\\ y\to y_0}} \frac{[f(x,y) - f(x_0, y_0)] - [A(x - x_0) + B(y - y_0)]}{\sqrt{(x - x_0)^2 + (y - y_0)^2}} = 0.$$

它们是函数 $f(x,y)$ 在点 $(x_0, y_0)$ 处可微的等价形式, 由它们都可得到 $f(x,y)$ 在点 $(x_0, y_0)$ 处可微, 且 $f_x(x_0, y_0) = A, f_y(x_0, y_0) = B$.

2. 可微性判定

(1) 必要条件: $f_x(x_0, y_0)$ 与 $f_y(x_0, y_0)$ 都存在;
(2) 充分条件: $f_x(x,y)$ 和 $f_y(x,y)$ 在 $(x_0, y_0)$ 连续;
(3) 用定义判定:
① $f_x(x_0, y_0)$ 与 $f_y(x_0, y_0)$ 是否都存在?
② 下式是否为零?
$$\lim\limits_{\substack{\Delta x \to 0\\Delta y\to 0}} \frac{[f(x_0 + \Delta x, y_0 + \Delta y) - f(x_0, y_0)] - [f_x(x_0, y_0)\Delta x + f_y(x_0, y_0)\Delta y]}{\sqrt{(\Delta x)^2 + (\Delta y)^2}}$$

3. 计算

若 $f(x,y)$ 可微, 则 $\text{d}z = \frac{\partial f}{\partial x} \text{d}x + \frac{\partial f}{\partial y} \text{d}y.$

五、连续、可导、可微的关系

第二节 偏导数与全微分的计算

1. 复合函数求导法

设 $ u = u(x, y), v = v(x, y) $ 可导，$ z = f(u, v) $ 在相应点有连续一阶偏导数，则
$$\frac{\partial z}{\partial x} = \frac{\partial f}{\partial u} \frac{\partial u}{\partial x} + \frac{\partial f}{\partial v} \frac{\partial v}{\partial x},$$ $$\frac{\partial z}{\partial y} = \frac{\partial f}{\partial u} \frac{\partial u}{\partial y} + \frac{\partial f}{\partial v} \frac{\partial v}{\partial y}.$$

2. 全微分形式不变性

设 $ z = f(u, v), u = u(x, y), v = v(x, y) $ 都有连续一阶偏导数。则
$$\text{d}z=\frac{\partial z}{\partial x} \text{d}x +\frac{\partial z}{\partial y}\text{d}y, \quad \text{d}z=\frac{\partial z}{\partial u}\text{d}u+\frac{\partial z}{\partial v}\text{d}v.$$

3. 隐函数求导法

(1) 由一个方程所确定的隐函数

  • 设 $F(x, y)$ 有连续一阶偏导数，且$F_y\neq 0$, 则 $y = y(x)$ 由 $F(x, y)=0$ 所确定, 且
    $$\frac{\text{d}y}{\text{d}x}=-\frac{F_x}{F_y}.$$

  • 设 $ F(x, y, z) $ 有连续一阶偏导数，且$F_z\neq 0$, 则 $z = z(x, y)$ 由 $ F(x, y, z) = 0 $ 所确定。

    • ① 公式法：$\frac{\partial z}{\partial x} = -\frac{F_x}{F_z}, \quad \frac{\partial z}{\partial y} = -\frac{F_y}{F_z};$
    • ② 等式两边求导法：$F_x + F_z \frac{\partial z}{\partial x} = 0, \quad F_y + F_z \frac{\partial z}{\partial y} = 0;$
    • ③ 利用微分形式不变性：$F_x \text{d}x + F_y \text{d}y + F_z \text{d}z = 0.$

(2) 由方程组所确定的隐函数（仅数学一要求）

  • 设$F(x,y,z),G(x,y,z)$有连续一阶偏导数，且
    $$J=\frac{\partial(F,G)}{\partial(y,z)}=\begin{vmatrix}F_y&F_z\ G_y&G_z\end{vmatrix}\neq 0$$则$y=y(x), z=z(x)$ 由$\begin{cases} F(x,y,z) = 0 \ G(x,y,z) = 0 \end{cases}$ 所确定。

  • 设$F(x,y,u,v),G(x,y,u,v)$有连续一阶偏导数，且
    $$J=\frac{\partial(F,G)}{\partial(u,v)}=\begin{vmatrix}F_u&F_v\G_u&G_v\end{vmatrix}\neq 0$$则$u = u(x,y), v = v(x,y)$ 由$\begin{cases} F(x,y,u,v) = 0 \ G(x,y,u,v) = 0 \end{cases}$ 所确定。

    • ① 等式两边求导法
      $$\begin{cases}
      F_x + F_u \frac{\partial u}{\partial x} + F_v \frac{\partial v}{\partial x} = 0, \
      G_x + G_u \frac{\partial u}{\partial x} + G_v \frac{\partial v}{\partial x} = 0;
      \end{cases}$$
    • ② 利用微分形式不变性
      $$\begin{cases}F_xdx + F_ydy + F_udu + F_vdv = 0, \G_xdx + G_ydy + G_udu + G_vdv = 0. \end{cases}$$

第三节 极值与最值

一、无条件极值

定义 设函数 $ z = f(x,y) $ 在点 $ P(x_0, y_0) $ 的某邻域内有定义，若对该邻域内任意的点 $ P(x,y) $ 均有$$f(x,y) \leq f(x_0, y_0) \quad (\text{或} \quad f(x,y) \geq f(x_0, y_0)),$$则称 $ (x_0, y_0) $ 为 $ f(x,y) $ 的极大值点（或极小值点）；称 $ f(x_0, y_0) $ 为 $ f(x,y) $ 的极大值（或极小值）。极大值点和极小值点统称为极值点；极大值和极小值统称为极值。

定理 1 （极值的必要条件）设 $ z = f(x,y) $ 在点 $ (x_0, y_0) $ 存在偏导数，且 $ (x_0, y_0) $ 为 $ f(x,y) $ 的极值点，则$$f_{x}(x_0, y_0) = 0, \quad f_{y}(x_0, y_0) = 0.$$

定理 2 （极值的充分条件）设 $z = f(x,y)$ 在点 $P_0(x_0,y_0)$ 的某邻域内有二阶连续偏导数，又 $f_{x}(x_0,y_0)=0,f_{y}(x_0,y_0)=0$。记$$A=f_{xx}(x_0,y_0),\quad B=f_{xy}(x_0,y_0),\quad C=f_{yy}(x_0,y_0).$$则有下述结论：

  1. 若 $AC-B^2>0$，则 $(x_0,y_0)$ 为 $f(x,y)$ 的极值点。
  • ① 若 $A<0$，则 $(x_0,y_0)$ 为 $f(x,y)$ 的极大值点；
  • ② 若 $A>0$，则 $(x_0,y_0)$ 为 $f(x,y)$ 的极小值点。
  1. 若 $AC-B^2<0$，则 $(x_0,y_0)$ 不为 $f(x,y)$ 的极值点。
  2. 若 $AC-B^2=0$，则 $(x_0,y_0)$ 可能为 $f(x,y)$ 的极值点，也可能不为 $f(x,y)$ 的极值点（此时一般用定义判定）。

求具有二阶连续偏导数的二元函数 $z=f(x,y)$ 极值的一般步骤:

  1. 求出 $f(x,y)$ 的驻点 $P_1\cdots,P_k$。

  2. 利用极值的充分条件判定驻点 $P_i(i=1,2,\cdots,k)$ 是否为极值点。

注：二元函数 $z=f(x,y)$ 在偏导数不存在的点也可能取到极值（如$f(x,y)=\sqrt{x^2+y^2}$）。

二、条件极值及拉格朗日乘数法

求 $z=f(x,y)$ 在条件$\varphi(x,y)=0$下的条件极值的一般方法为：
1. 构造拉格朗日函数 $$F(x,y,\lambda)=f(x,y)+\lambda\varphi(x,y);$$2. 求偏导数并构造方程组 $$\begin{cases}
f_x(x,y) + \lambda \varphi_x(x,y) = 0, \
f_y(x,y) + \lambda \varphi_y(x,y) = 0, \
\varphi (x,y) = 0.
\end{cases}$$解出 $x,y$ 及 $\lambda$，则其中 $(x,y)$ 就是函数 $f(x,y)$ 在条件 $\varphi(x,y)=0$ 下的可能的极值点。

推广：对于 $n$ 元函数在 $m$ 个约束条件下的极值问题，如求 $u=f(x,y,z)$ 在条件 $\varphi(x,y,z)=0, \psi(x,y,z)=0$ 下的极值，可构造拉格朗日函数$$F(x,y,z,\lambda ,\mu )=f + \lambda \varphi + \mu \psi.$$将 $F$ 对 $x,y,z,\lambda ,\mu$ 分别求偏导数，并构造方程组$$\begin{cases}
f_x(x,y,z) + \lambda \varphi_x(x,y,z) + \mu \psi_x(x,y,z) = 0, \
f_y(x,y,z) + \lambda \varphi_y(x,y,z) + \mu \psi_y(x,y,z) = 0, \
f_z(x,y,z) + \lambda \varphi_z(x,y,z) + \mu \psi_z(x,y,z) = 0, \
\varphi (x,y,z) = 0, \
\psi (x,y,z) = 0.
\end{cases}$$解出 $ x,y,z,\lambda$ 及 $\mu$，则其中 $(x,y,z)$ 就是可能的极值点。

注：对于实际问题，如果驻点唯一，且由实际意义知问题存在最大（小）值，则该驻点即为最大（小）值点。如果存在多个驻点，且由实际意义知道问题既存在最大值也存在最小值，只需比较各驻点处的函数值，最大的则为最大值，最小的则为最小值。

三、最大最小值

1. 求连续函数 $ f(x,y) $ 在有界闭域 $ D $ 上的最大最小值三部曲
(1) 求 $ f(x,y) $ 在 $ D $ 内部可能的极值点；
(2) 求 $ f(x,y) $ 在 $ D $ 的边界上的最大最小值；
(3) 比较。
